Basics Of CSS

What is css?

Ans : It is used to style your respective webpage.

Types of css :

  • Inline : written within the tag.
  • Internal : written anywhere within html page.
  • External : It is written seperately

Types of selectors :

  • Class
  • Id
  • Element
  • universal : body

  1. Inline styling :

    2. The style is an atribute not an element or tag.

  2. Internal styling :

    3. This is a element or tag. It can be written anywhere in the html page.

  3. External styling :

    4. we can create a external file and link that in head tag.Here, we created a p tag with two classes. Both have the same priority. so, that the second class property will override the first class property.

Note :

Priority for codes :
  1. inline : Always have the top most priority
  2. universal selector always have the second most priority.
  3. internal : If internal style comes last it hold the second priority otherwise it holds the last priority
  4. external : If external style comes last it hold the second priority otherwise it holds the last priority

Intermediate CSS

  1. Boxmodel :

    2. whatever we write it as a boxmodel.

    • margin : move towards to described position
    • border : move towards to described position
    • padding : move outwards to described position

  2. Positioning :

    3. Types :
    • static

      • top left right bottom are not working. It is placed where we written.

      Learn How it Works.

    • Relative

      • This is a relative position division.

      Learn How it Works.

    • Fixed

      • The position of my content will not move to any position

      It doesn't work in middle.

      Learn How it Works.

    • Absolute

      • Can't work independently. It depends on Relative position.

      Learn How it Works.

    • sticky

      • It fixed the content in a same place from where the content comes.

      Learn How it Works.
